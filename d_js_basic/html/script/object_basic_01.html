배열 이해하기

배열이란 목록의 나열이라는 의미를 가지고 있는 형식으로
이는 다양한 내용 및 변수 또는 함수를 나열하는 방식이다.


예시 : 커피의 종류

  <ul>
    <li>아메리카노</li>
    <li>아이스라떼</li>
    <li>카라멜마끼야또</li>
    <li>카푸치노</li>
    <li>프라푸치노</li>

객체 이해하기

배열과 유사하며, 나열의 의미는 같으나, 정의와 설명이 첨부된 형식
정의의 설명값이 함수로 처리되면 메소드라고 불리며, 단순설명으로 처리되면
프로퍼티라고 불리기도 함.

예시 : 음료의 종류

커피
커피원두를 주 원료로 한 음료
쥬스
다양한 종류를 가진 커피를 제외한 음료
디저트
커피 및 주스와 곁들여 막을 수 있는 냠
